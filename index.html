Welcome!

The Rebellion is a short choose your own adventure game. You can choose to play as an assassin or a mage. Work your way through the game; picking up items, killing enemies, and unlocking the secrets behind the rebel forces.
Below you will find information about the game and it's mechanics if you forget.

The game does not save if you navigate away from the page!

Classes

There are only two classes in the game: assassin and mage. Assassin start the game with five extra health and mages start the game with five extra strength.
